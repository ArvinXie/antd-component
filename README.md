# antd-components
> 对Antd组件进行二次封装和扩展，对业务系统更加友好

## Install 
```js
npm install antd-component 
 // or
yarn add antd-component
```

## Antd Version
4.x

### Use in your code

```javascript
import {
  CheckboxGroup,
  InputLable,
  InputRadio,
  InputRange,
  InputTag,
  SearchList,
  SearchTable,
  SearchToolBar,
} from "antd-component";
```
